# metropolia_javascript
Metropolia Javascript Assignments

## Modules and assignments

found in folders named mod-01, mod-02, etc. each module folder has assignment folders named t-01, t02, etc.

### init script

This repo contains bash script to automatically create module and assignment folders. It uses files from `mod-01/t-01/` as template. It does not overwrite existing folders.

```bash
./bin/init
```
